c++迭代器是一个对象，用于遍历容器中的每个元素。一个容器可能提供多种迭代器。例如，一个array 容器可能提供一个正向的迭代器，用于正向顺序遍历，另外提供一个反向的迭代器，用于反向遍历。一旦创建了合适类型的迭代器，程序员就可以使用迭代器提供的接口来遍历或者访问元素，而不用关注当前使用的是哪种遍历方式，也不用关注容器中存储的是哪种类型的数据。而且由于c++迭代器通常使用相同的接口进行遍历(使用++操作符访问下一个元素)和访问(使用*操作符访问当前元素)，所以可以使用一致的方法编译不同类型的容器。

#### 指针与迭代器
最简单的迭代器是指针，该指针（使用指针运算）适用于顺序存储在内存中的数据。 下面是一个简单数组遍历的例子：
```c++
#include <array>
#include <iostream>

int main() {
    std::array data{0,1,2,3,4,5,6,7};

    auto begin{&data[0]};
    auto end{begin + std::size(data)};
    // auto end{begin + sizeof(data)/sizeof(data[0])};
    for (auto ptr{begin}; ptr != end; ptr++) {
        std::cout << *ptr << ' ';
    } 
    std::cout << '\n';
    return 0;
}
```
从上面的例子可以总结出，如果需要遍历那么需要四个要素：
1. 开始位置`begin`
2. 结束位置`end`
3. 如果移动到一个位置`++操作符`
4. 获取当前元素的值`*操作符`
迭代器在c++标准库算法中应用非常广泛，后续章节将介绍几个常用的算法作用在迭代器上的情况。
